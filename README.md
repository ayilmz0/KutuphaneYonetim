# Kütüphane Yönetim Sistemi (RESTful API ve JWT ile Kimlik Doğrulama)

## Açıklama
Bu proje, kütüphane işlemlerini yönetmek için geliştirilmiş güvenli bir RESTful API sunmaktadır.
JWT tabanlı kimlik doğrulama ve rol tabanlı yetkilendirme ile kullanıcıların kitap yönetimi,
kullanıcı işlemleri ve ödünç/alma süreçlerini güvenli bir şekilde gerçekleştirmesi sağlanır.

## Teknolojiler
- C# / ASP.NET Core Web API  
- Entity Framework Core  
- MSSQL  
- JWT (JSON Web Tokens)  

## Özellikler
- JWT tabanlı kimlik doğrulama (kayıt/giriş sistemi)  
- Rol tabanlı yetkilendirme (Admin/Kullanıcı)  
- Kitap ve kullanıcı yönetimi  
- Kitap ödünç alma ve iade işlemleri  
- RESTful mimari ve iyi uygulama standartlarına uygun endpoint tasarımı  
- Katmanlı mimari: Controller → Service → Repository  
- Veri doğrulama ve hata yönetimi
